Writing/Documentation

I have written some basic Sega Game Gear programming tutorials using the SMSlib C library, available here.

I wrote a complete circuit analysis of the two switching regulators inside the Sega Game Gear handheld console in LaTeX. It remains the most complete documentation of these regulators and the power system inside this console in existence.

In 2020, I wrote a brief technical description (public Google Doc) of a combined DVD player/karaoke machine for the Texas Instruments Electronics Online Challenge. The DVD player belonged to a friend, and I wrote the document using only pictures of the device's internals. My writeup was voted into the top ten out of hundreds of submissions.

Contributions

I am the first person in the open-source community to investigate the integrated RISC-V microcontroller contained in the Rockchip RK3566 SoC, as used in the Pine64 Quartz64, PineTab2, and the PineNote. My research is ongoing and will help provide features to Pine64 devices.

I have contributed multiple commits to the official firmware repository for the Flipper Zero hacking multitool.

In 2022, I reverse engineered and documented the Bluetooth LE UART protocol used by the Sphero Mini educational robot.

Technical Projects

LAMDA

LAMDA is a 2-channel 100 kSPS 12-bit data sampling system built with an STM32F405 ARM microcontroller board. LAMDA captures multiple seconds of data at a time and saves it to an SD card, synchronizing samples against a pulse-per-second signal from a GPS unit. LAMDA is programmed with the STM32Cube framework.

alum

A complete file transfer package for the HP Reverse Polish Lisp calculator series, with a modern command-line interface. Alum features multiple serial protocols implemented in pure Rust.

mac-plus-serkey

This is a keyboard emulator for the Macintosh Plus and earlier models, allowing a modern computer to type on the Mac using an Arduino connected over USB. The computer captures keystrokes while the Arduino communicates with the Mac over its unique keyboard protocol.
