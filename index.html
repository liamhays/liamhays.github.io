Writing/Documentation/Reverse Engineering

I wrote a complete circuit analysis of the two switching regulators inside the Sega Game Gear handheld console in LaTeX. It remains the most complete documentation of these regulators and the power system inside this console.

I have written some basic Sega Game Gear programming tutorials using the SMSlib C library, available here.

I am the first person in the open-source community to investigate the integrated RISC-V microcontroller contained in the Rockchip RK3566 SoC, as used in the Pine64 Quartz64, PineTab2, and the PineNote. My research is ongoing and will help provide features to Pine64 devices.

I reverse engineered and documented the Bluetooth LE UART protocol used by the Sphero Mini educational robot.

In 2020, I wrote a brief technical description (public Google Doc) of a combined DVD player/karaoke machine for the Texas Instruments Electronics Online Challenge. The DVD player belonged to a friend, and I wrote the document without ever seeing it in person, using only pictures of the device's internals. My writeup was voted into the top ten out of hundreds of submissions.

Active Projects

Click on projects' names to be taken to their repositories for more information.

Flipwire

The first Flipper Zero Bluetooth software for Windows/Mac/Linux computers. Flipwire can transfer files and configure the Flipper remotely from a computer, features previously only available on the Flipper mobile app.

u-blox Flipper app

Flipper Zero app (available on the Flipper app store) for displaying data from u-blox GPS modules over I2C. u-blox can display data in multiple views, log to KML format, and align the Flipper's time to GPS time.

Quartz64 Model B case

A 3D-printable case for the Quartz64 Model B single-board computer designed in FreeCAD. STLs are available for using the Quartz with a microSD card or an NVMe drive.

Older Projects

These are projects I've done in the past that aren't receiving much attention right now, but hopefully are still useful and relevant.

Alum

A complete file transfer package for the HP RPL calculator series, with a modern command-line interfaces. Alum is written in pure Rust and requires no external dependencies on the host computer.

LAMDA

LAMDA is a 2-channel 100 kSPS 12-bit data sampling system built with an STM32F405 ARM microcontroller board. LAMDA captures multiple seconds of data at a time and saves it to an SD card, synchronizing samples against a pulse-per-second signal from a GPS unit. LAMDA is programmed with the STM32Cube framework.

mac-plus-serkey

This is a keyboard emulator for the Macintosh Plus and earlier models, allowing a modern computer to type on the Mac using an Arduino connected over USB. The computer captures keystrokes while the Arduino communicates with the Mac over its unique keyboard protocol.

Tunnel

A small Sega Game Gear game programmed in C with the devkitSMS framework. Get your ship to the end of the tunnel without hitting the walls or the enemies!
